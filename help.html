Troubleshooting

These additional steps are temporary while our app completes the certification process with Apple and Microsoft. We're working to simplify the installation process.

macOS: App Can't Be Opened

If you see a message that the app can't be opened because it's from an unidentified developer or that it's "damaged":

  1. Important: Make sure Fyenance.app is in your Applications folder (drag it there if it isn't)
  2. If you see a message saying the app is damaged and should be moved to trash, ignore it
  3. Open Terminal (use Search in top right)
  4. Copy and paste this command:
    xattr -c /Applications/Fyenance.app
  5. Try opening Fyenance again from Applications folder

The app must be in your Applications folder for these steps to work.

Windows Installation

SmartScreen Warning

If you see a "Microsoft Defender SmartScreen prevented an unrecognized app from starting" message:

  1. When the SmartScreen popup appears, click "More info"
  2. Click "Run anyway" at the bottom of the dialog
  3. If prompted for administrator rights, click "Yes"

This warning appears because we're a new application. The warning will be removed in time as our Microsoft reputation grows.

Administrator Rights Required

If you receive a permissions error or the app won't start:

  1. Find Fyenance in your Start menu
  2. Right-click on Fyenance
  3. Select "Run as administrator"
  4. Click "Yes" when prompted
